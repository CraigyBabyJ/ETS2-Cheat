ETS2 Cheat

This is the C# Windows app version.

Tool version: 1.0.0
ETS2 version: 1.60
Discord: https://discord.craigybabyj.com

Use:
  1. Start ETS2 and load your save.
  2. Run "ETS2 Cheat.exe".
  3. Click Enable to stop new damage.
  4. Click Reset Damage to clear existing truck damage.
  5. If the UI has not refreshed, switch screens or reopen the damage report.

Buttons:
  Enable/Disable - toggles the damage stopper code patch.
  Reset Damage   - zeros the truck's source damage and wear floats, including tyres.
  +100,000 Money - adds 100,000 euros to the current profile money.
  -100,000 Money - removes 100,000 euros from the current profile money.
  Refresh        - shows whether the game code is patched.
  Restore        - restores the original bytes.
